## Action Required — Items with Upcoming Deadlines

The following items require administrator action before their stated deadlines:

### 1. Secure Boot 2023 Certificate Readiness — **Deadline: June 2026**
**MC1293483** | Microsoft Defender for Endpoint | Platform Security

The Windows UEFI CA 2011 Secure Boot certificate expires in June 2026. Devices that have not been updated to trust the Secure Boot 2023 certificate will fail to boot securely and may be excluded from future protections. Use the new Secure Score recommendation in Microsoft Defender for Endpoint to identify affected devices and follow Microsoft's deployment guidance to update Secure Boot certificates and boot manager.

> **Action:** Review the Secure Score recommendation at [Microsoft Defender XDR](https://security.microsoft.com) and deploy Secure Boot 2023 certificate updates to all managed devices before June 2026.

---

### 2. Conditional Access: Resource Exclusion Enforcement — **Deadline: June 15, 2026**
**MC1223829** | Microsoft Entra | Identity & Access

Starting June 15, 2026, Conditional Access policies targeting "All resources" with resource exclusions will enforce for sign-ins requesting only OIDC or directory scopes (previously unenforced). Applications designed to request only these limited scopes may unexpectedly trigger MFA or device compliance challenges.

> **Action:** Identify custom applications that request only OIDC/directory scopes and verify they can handle Conditional Access challenges. Test before June 15, 2026. Most standard applications are already compliant.

---

### 3. Windows Autopatch Hotpatch Default Enablement — **Deadline: Before May 2026 Rollout**
**MC1248388** | Microsoft Intune | Endpoint Security

Starting May 2026, Windows Autopatch will automatically enable hotpatch security updates for eligible Intune-managed Windows devices. If your organization needs to delay or opt out, a tenant-level opt-out setting was made available on April 1, 2026. Devices require Virtualization-based Security (VBS) to receive hotpatches.

> **Action:** If opt-out is needed, configure the Autopatch tenant opt-out setting in the Intune admin center before the May 2026 enablement. Ensure eligible devices have VBS enabled to benefit from hotpatch when it activates.

---

### 4. SharePoint Records Management Features Retirement — **Ongoing from April 2026**
**MC1211579** | Microsoft Purview | Data Protection & Compliance

Microsoft has begun retiring SharePoint Online Information Management Policies, In-Place Records Management, and deletion-only document policies. Organizations relying on these legacy features must migrate to Microsoft Purview Data Lifecycle Management retention policies.

> **Action:** Audit SharePoint site collections for active Information Management Policies and In-Place Records Management configurations. Recreate equivalent retention policies in Microsoft Purview DLM and communicate changes to records management stakeholders.
