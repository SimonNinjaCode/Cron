## ⚠️ Action Required Items

### 1. External Access Tokens for Actionable Messages Retired — MC1189663
- **Deadline:** May 15, 2026 (enforcement has occurred)
- **Impact:** Any Outlook actionable message integration using external access tokens **now fails**. Update all integrations to use Microsoft Entra authentication immediately.
- **Link:** [MC1189663](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1189663)

### 2. Older MDE Mobile App Versions Retired — MC1276511
- **Deadline:** May 10, 2026 (enforcement has occurred)
- **Impact:** MDE mobile app versions released before February 2026 on iOS and Android have lost cloud connectivity. Verify all managed devices are running a post-February 2026 app version.
- **Link:** [MC1276511](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1276511)

### 3. SharePoint OTP Retirement — Entra B2B Migration Required — MC1243549
- **Ongoing deadline:** External sharing transitions to Entra B2B from May 2026; OTP authentication retirement begins July 2026; full enforcement **August 31, 2026**.
- **Impact:** External users without Microsoft Entra B2B guest accounts will lose access to previously shared links from July 2026. Create Entra B2B guest accounts for all required external users before July.
- **Link:** [MC1243549](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1243549)

### 4. "Require Approved Client App" Conditional Access Control Retires — MC1220751
- **Upcoming deadline:** June 30, 2026
- **Impact:** The `Require approved client app` grant control in Conditional Access will be unenforced after June 30, 2026, leaving policies without data protection. Migrate all affected policies to use `Require application protection policy` before this date.
- **Link:** [MC1220751](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1220751)
