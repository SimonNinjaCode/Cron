## ⚠️ Action Required Items

### 1. External Access Tokens for Actionable Messages Retired — MC1189663
- **Deadline:** May 15, 2026 (enforcement has occurred)
- **Impact:** Any Outlook actionable message integration using external access tokens **now fails**. Migrate all implementations to Microsoft Entra authentication immediately if not yet done.
- **Link:** [MC1189663](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1189663)

### 2. SharePoint OTP Retirement — Entra B2B Migration Required — MC1243549
- **Ongoing deadline:** OTP authentication retirement begins July 2026; full enforcement **August 31, 2026**.
- **Impact:** External users without Microsoft Entra B2B guest accounts will lose access to previously shared links from July 2026. Provision Entra B2B guest accounts for all required external users before July.
- **Link:** [MC1243549](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1243549)

### 3. Intune Data Warehouse Infrastructure Update — MC1188216
- **Deadline:** Before mid-June 2026 (update applies mid-June)
- **Impact:** Historical Data Warehouse data will be reset to ~30 days and the beta Power BI connector will be removed. Back up historical data and migrate reporting tools to the OData feed from the Intune admin center **before** the update applies.
- **Link:** [MC1188216](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1188216)

### 4. Conditional Access Enforcement for Resource Exclusions — MC1223829
- **Deadline:** June 15, 2026
- **Impact:** CA policies targeting All resources with exclusions will begin enforcing for OIDC/directory-scope sign-ins. Evaluate custom apps that request only these scopes and ensure they can handle MFA or device compliance challenges.
- **Link:** [MC1223829](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1223829)

### 5. "Require Approved Client App" Conditional Access Control Retires — MC1220751
- **Deadline:** June 30, 2026
- **Impact:** The `Require approved client app` grant control becomes read-only and stops being enforced after June 30, 2026. Migrate all affected CA policies to use `Require application protection policy` before this date.
- **Link:** [MC1220751](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1220751)

### 6. Secure Boot Certificate Transition — MC1293483
- **Deadline:** June 2026 (Windows UEFI CA 2011 certificate expiry)
- **Impact:** Devices not updated to Secure Boot 2023 certificates may lose firmware security protections. Review the MDE Secure Score recommendation to identify unprepared devices and apply the certificate update.
- **Link:** [MC1293483](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1293483)
