## ⚠️ Action Required Items

### 1. Older MDE Mobile App Versions Retired — MC1276511
- **Deadline:** May 10, 2026 (enforcement has occurred)
- **Impact:** MDE mobile app versions released before February 2026 on iOS and Android have lost cloud connectivity. Verify all managed devices are running a post-February 2026 app version; push updates via Intune where needed.
- **Link:** [MC1276511](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1276511)

### 2. External Access Tokens for Actionable Messages Retired — MC1189663
- **Deadline:** May 15, 2026 (enforcement has occurred)
- **Impact:** Any Outlook actionable message integration using external access tokens **now fails**. Migrate all implementations to Microsoft Entra authentication immediately if not yet done.
- **Link:** [MC1189663](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1189663)

### 3. SharePoint OTP Retirement — Entra B2B Migration Required — MC1243549
- **Ongoing deadline:** New invitations use Entra B2B from May 2026; OTP authentication retirement begins July 2026; full enforcement **August 31, 2026**.
- **Impact:** External users without Microsoft Entra B2B guest accounts will lose access to previously shared links from July 2026. Identify all external OTP users and provision Entra B2B guest accounts before July.
- **Link:** [MC1243549](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1243549)

### 4. "Require Approved Client App" Conditional Access Control Retires — MC1220751
- **Upcoming deadline:** June 30, 2026
- **Impact:** The `Require approved client app` grant control in Conditional Access becomes read-only and unenforced after June 30, 2026. Migrate all affected policies to use `Require application protection policy` before this date to maintain data protection.
- **Link:** [MC1220751](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1220751)

### 5. Secure Boot 2023 Certificate Transition — MC1293483
- **Upcoming deadline:** June 2026 (Windows UEFI CA 2011 certificate expiry)
- **Impact:** Devices not updated with Secure Boot 2023 certificates may fail to boot or lose firmware security protections after June 2026. Review the new MDE Secure Score recommendation to identify unprepared devices and apply updates.
- **Link:** [MC1293483](https://admin.microsoft.com/Adminportal/Home#/MessageCenter/:/messages/MC1293483)
